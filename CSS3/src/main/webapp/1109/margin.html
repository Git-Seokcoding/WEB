margin - 요소의 바깥 여백

1. margin의 방향별 지정법

  • margin-top:n; 위에만 적용
  • margin-right:n; 우측에만 적용
  • margin-bottom:n; 아래만 적용
  • margin-left:n; 좌측에만 적용

2. margin의 값 갯수별 지정법

  • margin:n ; 양사방 동일하게 적용
  • margin:n n ; 상하와 좌우가 서로 동일하게 적용
  • margin:n n n; 상, 좌우 서로 동일, 하 순으로 적용
  • margin:n n n n; 상,우,하, 좌 순으로 적용
