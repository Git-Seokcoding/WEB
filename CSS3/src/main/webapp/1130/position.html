position - 요소의 위치 지정법

  • position:static; (기본) 위치 변경 불가
  • position:relative; 현재 위치를 기준으로 ~
  • position:absolute; 상위요소를 기준으로 ~
  • position:fixed; 브라우저를 기준으로 고정
  • position:sticky; relative와 fixed의 믹스

거리 지정법

  • left:n;
  • right:n;
  • top:n;
  • bottom:n;

5. position:sticky;

sticky 적용할 대상은 상위요소가 없어야 함

1. position:static; (기본) 위치 변경 불가

position:static; left: 1000px;

2. position:relative; 현재 위치 기준으로 ~

position:relative;
left: 1000px;
position:relative;
right: 1000px;
position:relative;
top: 250px;
position:relative;
bottom: 250px;
position:relative;
left: 30em; bottom: 5em;
position:relative;
left: 800px; top: 120px;

3. position:absolute; 상위요소 기준으로 ~ (기준이 될 상위요소는 position: relative;로 지정)

position:absolute;
left: 1000px;
position:absolute;
right: 1000px;
position:absolute;
top:20px;
position:absolute;
bottom: 100px;
position: absolute; right: 0; bottom: 0;
position: absolute; left: 50%; top:50%;
