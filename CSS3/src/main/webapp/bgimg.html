background - 배경

  1. background-color:color값; 배경 색상
  2. background-image:url("이미지경로"); 배경 이미지

background-image 2개 지정

background-image:url("이미지-1 경로"), url("이미지-2 경로");
( 기타 세부속성들도 따로 지정하려면 ,를 추가 background-repeat:repeat, no-repeat ; )

background-image의 속성들

  • background-repeat: ; 반복 지정
    1. repeat (기본) 양사방 반복
    2. repeat-x 가로로만 반복
    3. repeat-y 세로로만 반복
    4. no-repeat 반복 안함
  • background-size: ; 크기
    1. cover 이미지 하나로 영역 전체를 다덮기 (이미지가 잘려나갈 수도)
    2. contain 최대한 이미지로 영역 전체를 덮기 (항상 이미지 온전하게 나옴)
    3. Xn Yn 가로 세로 크기 직접 지정
  • background-position: ; 위치
    1. left/center/right top/center/bottom 조합 지정
      (기본 left top = 0 0)
    2. Xn Yn 가로 세로 위치 직접 지정
  • background-attachment: ; 스크롤 여부
    1. scroll (기본) 스크롤
    2. fixed 고정
    3. local

background-attachment: ; 스크롤 여부

[ 특정영역에 스크롤바 생성시 ] scroll (기본) 자신 영역에는 고정, 브라우저에는 스크롤.
fixed 자신 영역과 브라우저 모두 고정.
local 자신 영역과 브라우저 모두 스크롤.
