(Background) gradient - (배경) 그라데이션

gradient의 종류

  1. Linear Gradient 선형 그라데이션
  2. Radial Gradient 방사형 그라데이션
  3. Conic Gradient 원뿔형 그라데이션

gradient의 반복(패턴)

  • repeating-linear-gradient(A,B)
  • repeating-radial-gradient(A,B)
  • repeating-conic-gradient(A,B)

1. Linear Gradient 선형 그라데이션

background:linear-gradient(red,yellow);

background: linear-gradient(pink,olive,violet,navy);

background: linear-gradient(120deg,olive,pink,navy);

background: linear-gradient(120deg,olive 20%,pink 80%,navy);

2. Radial Gradient

background:radial-gradient(red,yellow);

background:radial-gradient(circle,pink,olive);

background: radial-gradient(circle,pink 39%,olive 40%);

3. Conic Gradient

background: conic-gradient(olive,pink);

background: conic-gradient(olive 30deg,pink 50deg);
