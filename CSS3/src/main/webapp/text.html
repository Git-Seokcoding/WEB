text - 로 시작하는 글자 속성들

  • text - align : ; 글자 수평 정렬
  • text - decoration: ; 글자의 줄들
  • text - indent: ; 글자 들여쓰기
  • text - transform: ; (영어)대소문자 변경
  • text - shadow: ; 글자의 그림자
  • text - overflow: ; 영역밖의 글자
